Pokémon Type Chart

This tool will break down the weaknesses, strengths, and other interactions between all the different pokémon types. Simply select the type you want to learn about from the dropbox and push the "Submit" button. The graph on the right will automatically tell you the different interactions it has with the other types.
Key;

  • Strengths - What types are weak against this type
  • Ineffective - What types resist this type
  • Futilities - What types are completely immune to this type
  • Weaknesses - What types this type is weak to
  • Resistances - What types aren't very effective against this type
  • Immunities - What types this type are immune to
