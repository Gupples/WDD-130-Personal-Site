Pokémon Type Chart

Pokémon types are like Rock, Paper, Scizzors. Each type is good against some types and weak against others. This tool will break down the weaknesses, strengths, and other interactions between all the different pokémon types. Simply select the type you want to learn about from the dropbox and push the "Submit" button. The graph on the right will automatically tell you the different interactions it has with the other types.
DISCLAIMER; type icons were found in Creative Commons Google search, are copyright-free and NOT accurate to the icons used in the actual games.

Key;

  • Strengths - What types are weak against this type
  • Ineffective - What types resist this type
  • Futilities - What types are completely immune to this type
  • Weaknesses - What types this type is weak to
  • Resistances - What types aren't very effective against this type
  • Immunities - What types this type are immune to
